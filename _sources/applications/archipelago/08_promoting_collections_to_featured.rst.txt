=======================================
Promoting Collections on the Front Page
=======================================

------------
Requirements
------------

* Go to Configuration → Search API → Drupal
* Click the Fields tab
* Look for Promoted to front page (promote). If it's not listed, click Add fields and add it:
    * label: :code:`Promoted to front page`, machine name: :code:`promote`, type :code:`Integer`
Save, then re-index the index
