====================================
Changing the template for a worktype
====================================

1. Go to :code:`admin/config/archipelago/viewmode_mapping`.

2. Add a new mapping. Select "Book" for ADO Type and "Digital object with Mirador viewer" for View Mode. Click add.

3. The new Book template should have shown up at the bottom of the list. Drag it to the previous Book template. Uncheck the box next to the old template.

4. Click save configuration.
